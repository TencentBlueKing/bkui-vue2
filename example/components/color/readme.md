## Color 色彩

`bk-magic-vue` 使用以下调色板的颜色作为开发与设计规范，为构建的产品提供一致的外观视觉感受。颜色主要分为主色、辅助色、中性色三部分，主色部分为蓝鲸系列产品使用的主题色；而辅助色主要适用于状态类标识，也会用到个别需要特殊处理的元素；中性色主要使用在文字、边框、线条、背景等等。

### 主色 {page=#/color}
产品主色，蓝色代表严谨、理性和科技。
